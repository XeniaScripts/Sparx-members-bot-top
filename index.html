Discord Member Authorization Server

This server handles the OAuth2 process for the Sparx bot and connects to the PostgreSQL database.

Instructions

To authorize, please use the /authorize slash command in the Discord server. The bot will provide the correct link which uses the /authorize endpoint on this server.

This static page helps ensure the Vercel deployment runs correctly by providing a stable root (`/`) path.
